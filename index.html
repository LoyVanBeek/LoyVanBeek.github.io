    Welcome to Loy's GitHub Pages.

    Hi there, my name is Loy. On these pages, I (will) post about the stuff I do. So, what do I do? Programming mostly. I work for Sioux Embedded Systems and in my free time I work on some hobby projects, such as making the Amigo robot do cool stuff. Scouting is another hobby, which I sometimes combine with programming.

    Most of my personal projects are on my GitHub page. Some examples: a computer vision program that can play the Set card game, a system where kids in my scout group use a QR code to see what activities are planned for them and some ROS interfaces to Natural Language Processing (NLP) web APIs like Maluuba and Wit.ai (wrappers at https://github.com/LoyVanBeek/wit_ros and https://github.com/LoyVanBeek/maluuba_ros)

    The languages I use most frequently are Python, C# and lately Prolog. I'd love to learn Lisp once I find a good application of it.
